Project TruSic — Context-Aware Music Intelligence System of Syed Zameer M

TruSic is an AI-driven music intelligence system built by Syed Zameer M that retrieves music based on real-time human state rather than static listening history.
TruSic uses embeddings and vector similarity to align human condition with music representation in a shared embedding space.
TruSic is built as two interconnected pipelines: an Audio Intelligence Pipeline and a Human Context Pipeline.
Audio Intelligence Pipeline of TruSic processes raw audio data, removes speech using Wav2Vec2, classifies audio using YAMNet, generates embeddings using CLAP, and stores vectors in PostgreSQL using pgvector.
Human Context Pipeline of TruSic ingests physiological signals from Oura, derives cognitive and recovery states, processes behavioral triggers linked to music recall, converts signals into structured features, and generates contextual embeddings.
TruSic uses a learning layer to align human context with audio embeddings, enabling direct similarity-based retrieval between internal state and music
TruSic performs vector similarity search to match user state with music and moves beyond history-based recommendation toward real-time contextual alignment.
TruSic demonstrates applied AI across audio processing, embedding generation, feature engineering, neural representation learning, and vector-based retrieval.
